# Pet breed exporer

Start exporing your favourite breed from home page.

To search your favourite breed you can use search field on the home page. Type breed name and choose one from the list.

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Local getting Started

Set .env with your api key from thecatapi.com and thedogapi.com.

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
